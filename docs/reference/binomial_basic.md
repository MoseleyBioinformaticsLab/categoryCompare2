# do binomial test

does a binomial test

## Usage

``` r
binomial_basic(
  positive_cases,
  total_cases,
  p_expected = 0.5,
  direction = "two.sided",
  conf_level = 0.95
)
```

## Arguments

- positive_cases:

  number of positive instances

- total_cases:

  total number of cases observed

- p_expected:

  what is the expected probability

- direction:

  which direction is the test

- conf_level:

  confidence level for the confidence interval

## Value

list
