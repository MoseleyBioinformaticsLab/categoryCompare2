# get statistics

extract all statistics for a
[`statistical_results`](https://moseleybioinformaticslab.github.io/categoryCompare2/reference/statistical_results-class.md)
object. These can then be combined into a `data.frame` that can be
returned or used to annotate the graph of annotations.

## Usage

``` r
extract_statistics(in_results)

# S4 method for class 'statistical_results'
extract_statistics(in_results)
```

## Arguments

- in_results:

  the
  [`statistical_results`](https://moseleybioinformaticslab.github.io/categoryCompare2/reference/statistical_results-class.md)
  object

## Value

data.frame
