# convert gsea object

Takes an \`gsea_result\`, and converts it to the table expected by
\`fgsea\`. This should only be done on those that have \`gsea\` as the
\*Enrichment Method\*.

## Usage

``` r
gsea_result_to_fgsea(in_gsea)
```

## Arguments

- in_gsea:

  the gsea_result object

## Value

data.table
