# combine text

Given lists of named character objects, and a character vector of names
to be in the final object, either get the character string from the list
that has the names, or check that the character string is the same
across all of the lists.

## Usage

``` r
combine_text(list_characters, names_out, text_id)
```

## Arguments

- list_characters:

  list containing named character strings

- names_out:

  the full list of names to use

- text_id:

  what is the name for that thing being put out

## Value

named character vector
