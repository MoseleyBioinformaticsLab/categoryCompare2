# convert enriched object

Takes an \`enriched_result\`, and converts it to the table expected by
\`fgsea\`. This should only be done on those that have \`gsea\` as the
\*Enrichment Method\*.

## Usage

``` r
enriched_to_fgsea(in_enriched)
```

## Arguments

- in_enriched:

  the enrichment object

## Value

data.table
