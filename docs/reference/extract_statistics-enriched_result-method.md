# extract statistics

extract single set of statistics from a
[`enriched_result`](https://moseleybioinformaticslab.github.io/categoryCompare2/reference/enriched_result.md)
object and create a `data.frame`.

## Usage

``` r
# S4 method for class 'enriched_result'
extract_statistics(in_results)
```

## Arguments

- in_results:

  the
  [`enriched_result`](https://moseleybioinformaticslab.github.io/categoryCompare2/reference/enriched_result.md)
  object

## Value

data.frame
